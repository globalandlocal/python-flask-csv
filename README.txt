Попытка в онлайн сервис для работы с CSV
Перед работой в view.py неободимо переназначить url на любую другую папку,в ней будут сохраняться загруженные,созданные файлы
Как пользоваться:
0. запустить main.py.
1.открыть http://127.0.0.1:5000/

2.на вкладке Search  идет поиск по тексту внутри файла,для этого загружаете файл.Будут выведены первые 20 совпадений.
в 1 поле вводите назнание стоблца , по которому необходимо искать,
во 2  поле  вводите текст,который нужно найти.

Если не помните названия столбцов просто отправьте файл без фильтров.

3.На вкладке Delete работа происходит  аналогично работы с Search.
Но будет  создан новый файл где будет сохранена вся информация ,
кроме тех строк где встречается указанный текст.
Информация которая не была записана в новый файл будет выведена на экран.

вложен тестовый posts.csv с которым осуществлялась работа.
Пример:
загрузить posts.csv
справа написать text
внизу написать Начало
(идет регистронезависимый поиск)
результат на экране.
